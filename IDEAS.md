# Roadmap: функциональные модели вне текущего движка

Текущий генератор строит watertight-геометрию «сборкой оболочек» (примитивы +
вложенные проникающие shell'ы, без CSG, без сплайнов, без винтовой геометрии).
Ниже — функциональные классы, требующие новой возможности. Отсортировано по
приоритету «спрос / сложность / совместимость с watertight-подходом».

## 🥇 1. Резьбовые соединения — *helix-sweep* — ГОТОВО (v73–v74)
- ✅ Крышка (внутренняя резьба + насечка), штуцер (наружная резьба на фланце),
  **банка/бутыль** (полый корпус + резьбовое горло). Крышка↔горло свинчиваются по построению.
- ✅ Watertight: резьба как heightfield на цилиндре r(θ,y) = base ± depth·profile(frac(y/P − hand·starts·θ/2π)),
  торцы закрываются веером — тот же приём, что sheet-texture.
- Осталось (по желанию): заходная фаска (lead-in), чтобы резьба легче стартовала; уплотнительное
  кольцо/бортик; привязка резьбового горла к произвольным полым контейнерам (round/squircle).

## 🥈 2. Print-in-place механика — *контролируемые зазоры между несоединёнными телами*
- Живые петли, коробки с интегрированной крышкой на петле (печать в сборе).
- Snap-fit защёлки, клипсы, кабель-стяжки, разъёмные хомуты для труб/кабеля.
- Гэп: НЕ union — уметь оставлять раздельные watertight-компоненты в одном файле
  с зазором 0.2–0.4 мм.

## 🥉 3. Зубчатые передачи — *эвольвентный профиль*
- Шестерни spur / helical, рейки, звёздочки GT2 под ремень, храповики.
- Плоский профиль → экструзия, watertight простой. Большой спрос.

## 4. Крепёж и монтаж — *зенковки / резьбовые бобышки (сейчас нет булевых вычитаний)*
- Настенные кронштейны, VESA-адаптеры, держатели инструмента.
- Бобышки под винт / гайку-вставку (heat-set), зенкованные отверстия под потай.

## 5. TPMS-инфилл как функция — *marching cubes по неявному полю*
- Гироид/Schwarz-P проставки, демпферы, энергопоглотители, лёгкие несущие блоки
  заданной жёсткости. Обобщение существующей логики паттернов на 3D.

## Отложенный декор (по запросу пользователя исключён из приоритета)
- Тела вращения (вазы/кашпо), литофания (фото→панель), органика/скульпт.
